##  Descripción
Este proyecto es un taller introductorio sobre los conceptos básicos de la Programación Orientada a Objetos (POO), donde se explican principios fundamentales como encapsulación, clases y objetos.

---

##  Objetivo
El objetivo de este proyecto es reforzar el aprendizaje de POO mediante ejemplos prácticos y una presentación interactiva.
